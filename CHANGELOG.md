## [Unreleased]

### Added
- Initial public release.
- Dashboard with net worth, cash-flow, savings rate, runway and asset allocation.
- Multi-account model (checking, savings, cash, investment, real estate, credit).
- Manual transaction entry (income / expense / transfer).
- Recurring transactions with auto-application engine.
- Net worth and cash-flow forecasting up to 60 months.
- Budgets per category with the 50 / 30 / 20 rule overview.
- Local-first AI assistant powered by Deepseek-R1 14B via Ollama.
- On-device speech-to-text powered by WhisperKit (Whisper Large-v3) — forced French language.
- Block-aware Markdown renderer for assistant replies.
- JSON persistence for conversation history.
- Build script that produces a `.app` bundle without requiring Xcode.
