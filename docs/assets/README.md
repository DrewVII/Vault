# Marketing assets

Place screenshots and GIFs of the app here, then reference them from the top-level `README.md`.

Suggested captures:

| File              | What to show                                                |
|-------------------|-------------------------------------------------------------|
| `dashboard.png`   | The main dashboard with KPIs and charts populated           |
| `accounts.png`    | The accounts grid with multiple account kinds               |
| `forecast.png`    | A 24-month forecast curve                                   |
| `assistant.gif`   | A short loop showing a question being answered with the `<think>` panel expanded |
| `voice.gif`       | The microphone button capturing a French sentence and the transcript appearing |

Rules of thumb for great repo screenshots:

- Use a dark or light desktop background that does not clash with the app.
- Capture at 2× / Retina, then `pngquant` to slim files.
- For GIFs, target ≤ 5 MB; tools: [Kap](https://getkap.co/), [Gifski](https://gif.ski/).
